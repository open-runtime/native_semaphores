# runtime_native_semaphores v1.0.5

> Bug fix release — 2026-02-24

## Bug Fixes

- **CI Configuration Updates** — Updated the CI matrix to use organization-managed runners (`runtime-*`) as defaults for Linux and Windows platforms. Adjusted GitHub token injection in Git configurations to improve reliability, and switched the `runtime_ci_tooling` dependency URL to HTTPS for better CI compatibility, bumping it to `^0.13.0`.
